LazyJSON Demos

Here are a few quick examples of plugin usage. We'll utilize a 'database' of members of the Baggins family... turns out Bilbo had quite a few cousins. Take a look at the included api.php to see how the JSON is processed and returned.

Buttons & API Pagination Show Code 

				$( 'ul#buttons' ).lazyjson({
			        	childEl: 'li',
			        	loaderImg: 'img/loader.gif',
			        	pagination: {
			            		active: true,
			            		pages: 1,
			            		count: 10
			        	},
			        	api: {
			            		uri: 'api.php',
			            		pagination: true
			        	}
				});
  • {{id}} - {{data.first_name}} {{data.last_name}}

Using A Custom Event Show Code 

				$( 'ul#custom_event' ).lazyjson({
				        childEl: 'li',
				        loaderImg: 'img/loader.gif',
				        pagination: {
					            active: true,
					            pages: 1,
					            count: 10,
					            loadOnEvent: 'click',
					            loadOnTarget: 'list-more'
				        },
				        api: {
				    	        uri: 'api.php'
				        }
				});
